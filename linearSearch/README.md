## Linear Search

<p align="center">
    <img src="../../.github/linearsearch.gif">
</p>

Linear Search is a search based on a iteration in each element of a collection. It is very intuitive and its performance will depends on the size of the entry array.

#### Complexity 
##### Worst Cases: O(n)
##### Medium Cases: O(n/2)
##### Best Cases: O(1)
